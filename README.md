# Portfólio - Gerson Bruno

Este é o portfólio de estudos de **Gerson Bruno**, focado em **Front-end**.  
O projeto reúne páginas e seções desenvolvidas para prática de HTML, CSS e JavaScript, incluindo:

- Seção **Home** com apresentação pessoal e links de redes sociais.
- Seção **Sobre** com informações pessoais e trajetória de estudos.
- Seção **Skills** mostrando habilidades técnicas e experiência prática.
- Seção **Work** com exemplos de projetos e trabalhos realizados.
- Seção **Contato** com formulário funcional usando **EmailJS**.

## Tecnologias utilizadas

- HTML5
- CSS3 / SCSS
- JavaScript
- Boxicons
- EmailJS para envio de mensagens do formulário

## Observações

Este portfólio é **um projeto de estudo e prática**, com foco em desenvolver e testar conceitos de front-end, layout responsivo e interatividade com JavaScript.

## Como visualizar

Basta abrir o arquivo `index.html` no navegador ou hospedar o projeto usando GitHub Pages.
